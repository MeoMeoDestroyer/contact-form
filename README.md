# Kate Nguyen – Professional Network & Portfolio

A full-stack web application built as part of the Software Development program at Green River College. The app serves as a personal resume site, portfolio showcase, and professional networking tool with a contact/guestbook system backed by a MySQL database.

## 🌐 Deployed URL

[http://64.23.137.40:3005/home](http://64.23.137.40:3005/home)

## 📋 Project Description

This application combines a personal resume, portfolio page, and contact form into a single cohesive system. Visitors can browse projects, submit a contact form, and optionally join a mailing list. Submissions are stored in a MySQL database and viewable through an admin dashboard.

### Features
- Resume/homepage with education, skills, and experience
- Portfolio page showcasing SDEV 301 and SDEV 305 projects
- Contact form with both client-side and server-side validation
- Mailing list opt-in with email format preference
- Admin dashboard with sortable/searchable submissions table (DataTables)
- Confirmation page after successful form submission

## 🛠️ Technologies Used

- Node.js & Express.js
- MySQL 8 / mysql2
- EJS templating with reusable partials
- DataTables.js
- dotenv
- DigitalOcean App Platform
